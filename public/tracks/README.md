# IELTS Audio Tracks

Bu papkada IELTS training audio fayllari saqlanadi.

## Struktura

```
tracks/
├── listening/
│   └── default.mp3 (placeholder)
├── reading/
│   └── default.mp3 (placeholder)
├── writing/
│   └── default.mp3 (placeholder)
└── speaking/
    └── default.mp3 (placeholder)
```

## Qo'llanma

1. **default.mp3** - vaqtinchalik placeholder
2. Yangi audio admin panel orqali yuklanadi
3. Audio nomi: `{timestamp}_{originalName}.mp3`
4. Training yaratishdan OLDIN audio yuklanishi kerak

## Audio Format

- Format: MP3
- Max size: 50MB
- Sample rate: 44.1kHz yoki 48kHz
- Bitrate: 128kbps - 320kbps
